# Bird Call UMAP

This is an experimental project for bird call clustering analysis using UMAP.

## Key Features
- Clustering analysis of bird calls
- Visualization using UMAP
- Support for various audio formats

## Requirements
- Python 3.x
- NumPy
- SciPy
- UMAP

## Current Limitations
- Limited dataset
- Ongoing improvements in clustering algorithms
- Work-in-progress project
